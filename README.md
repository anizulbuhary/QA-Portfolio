# Fathima Anizul | SQA Portfolio

A premium, interactive portfolio website for a Software Quality Assurance (SQA) Engineer. Built with React, Vite, Framer Motion, and TypeScript.

## 🚀 Getting Started

To run this project locally, follow these steps:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your system.

### 2. Installation
Extract the ZIP file, open your terminal in the project folder, and run:
```bash
npm install
```

### 3. Run Development Server
Start the interactive development environment:
```bash
npm run dev
```
Once the command finishes, open the local URL (usually `http://localhost:5173`) in your browser.

## 🕹️ Interactive Features
- **Bug Interaction**: Click/Hover on the animated bugs to "resolve" them! (Capped at 8 bugs max for performance).
- **SQA Stats**: Real-time monitoring of "Defect Density" and resolution counts.
- **Japandi Design**: Modern, calm, and minimalist aesthetic with premium animations.

## 🛠️ Project Structure
- `src/components`: UI components and page layouts.
- `src/pages`: Main page views (About, Experience, Projects, etc.).
- `src/context`: Global state management for bug interactions.
- `public`: Static assets including the custom branding favicon.
